# aws-s3-static-website

This project demonstrates static website hosting with Amazon S3 and basic cloud documentation.

## Project Goal
Build and publish a simple static website using AWS S3 while documenting the setup in GitHub.

## Why I Built This
I am building this project as part of my AWS/cloud learning journey. The goal is to gain hands-on experience with static website hosting, basic frontend structure, and project documentation that can be shown in a professional portfolio.

## Skills Practiced
- Amazon S3
- Static website hosting
- HTML
- CSS
- GitHub repository setup
- Cloud project documentation

## Project Files
- `index.html` - Main website page
- `style.css` - Styling for the website
- `docs/notes.md` - Project notes and lessons learned
- `screenshots/` - Proof of setup and deployment

## Project Status
** Live at https://paul-giff.com **

## Planned Build Steps
1. Create the website files locally/in GitHub
2. Build a simple static page
3. Create an S3 bucket
4. Upload the website files to S3
5. Enable static website hosting
6. Review bucket permissions
7. Test the website endpoint
8. Document the deployment with screenshots

## Lessons I Expect to Learn
- How static hosting works in AWS
- How to organize a beginner cloud project in GitHub
- How to document technical work clearly for employers and recruiters
